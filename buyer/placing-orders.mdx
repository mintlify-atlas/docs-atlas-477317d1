## Overview

Placing an order on CURaise involves browsing a fundraiser's items, adding them to your cart, and completing the checkout process. This guide covers the entire order placement workflow.

## Viewing Fundraiser Items

After selecting a fundraiser from the browse page, you'll see the fundraiser detail page at `/buyer/fundraiser/{id}` with:

- **Fundraiser Information**: Name, description, hosting organization
- **Pickup Details**: All scheduled pickup events with dates, times, and locations
- **Announcements**: Updates from fundraiser organizers
- **Items Grid**: Available products for purchase
- **Cart Sidebar**: Real-time cart summary (desktop only)

<Note>
The page fetches data using two API calls:
- `/fundraiser/{id}` - Returns `CompleteFundraiserSchema` with full details
- `/fundraiser/{id}/items` - Returns array of `CompleteItemSchema`

Source: `frontend/src/app/buyer/fundraiser/[id]/page.tsx:37-42`
</Note>

## Adding Items to Cart

### Cart State Management

CURaise uses Zustand with localStorage persistence for cart management:

```typescript
interface CartState {
  // Map fundraiser IDs to cart items
  carts: Record<string, CartItem[]>;
  
  addItem: (fundraiserId: string, item: CompleteItemSchema, quantity: number) => void;
  removeItem: (fundraiserId: string, item: CompleteItemSchema) => void;
  updateQuantity: (fundraiserId: string, item: CompleteItemSchema, quantity: number) => void;
  clearCart: (fundraiserId: string) => void;
}

export const CartItem = z.object({
  item: CompleteItemSchema,
  quantity: z.number().int().nonnegative(),
});
```

```typescript

    token,
    body: dataToSubmit,
  });
  redirect("/buyer/order/" + result.data.id + "/submitted?fromCheckout=true");

  toast.error(error instanceof Error ? error.message : "Failed to create order");
  setIsSubmitting(false);
}
```

## Best Practices

<Accordion title="Quantity Management">
  When quantity is reduced to 0, the item is automatically removed from the cart:
  
  ```typescript
  const handleQuantityChange = (item: CartItem["item"], delta: number) => {
    const currentQuantity = cart.find((ci) => ci.item.id === item.id)?.quantity || 0;
    const newQuantity = Math.max(0, currentQuantity + delta);
    
    if (newQuantity === 0) {
      removeItem(fundraiser.id, item);
    } else {
      updateQuantity(fundraiser.id, item, newQuantity);
    }
  };
  ```

</Accordion>
