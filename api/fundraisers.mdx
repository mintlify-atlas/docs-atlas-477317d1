## List Fundraisers

<CodeGroup>
```bash cURL
curl https://api.curaise.com/api/fundraiser
```

```typescript TypeScript
const response = await fetch('https://api.curaise.com/api/fundraiser')
const data = await response.json()
```
</CodeGroup>

**GET** `/api/fundraiser`

Retrieves all published fundraisers.

### Authentication

Public endpoint - no authentication required.

### Response

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="data" type="Fundraiser[]">
  Array of fundraiser objects
  
  <Expandable title="Fundraiser Object">
    <ResponseField name="id" type="string (uuid)">
      Unique fundraiser identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Fundraiser name (1-255 characters)
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Detailed description
    </ResponseField>
    
    <ResponseField name="published" type="boolean">
      Whether fundraiser is publicly visible
    </ResponseField>
    
    <ResponseField name="venmoUsername" type="string | null">
      Venmo username for payments (5-30 characters)
    </ResponseField>
    
    <ResponseField name="venmoEmail" type="string | null">
      Email associated with Venmo account
    </ResponseField>
    
    <ResponseField name="goalAmount" type="string (decimal) | null">
      Target fundraising amount
    </ResponseField>
    
    <ResponseField name="buyingStartsAt" type="string (date)">
      ISO 8601 timestamp when buying period begins
    </ResponseField>
    
    <ResponseField name="buyingEndsAt" type="string (date)">
      ISO 8601 timestamp when buying period ends
    </ResponseField>
    
    <ResponseField name="imageUrls" type="string[]">
      Array of image URLs
    </ResponseField>
    
    <ResponseField name="organization" type="Organization">
      Associated organization (BasicOrganizationSchema)
    </ResponseField>
    
    <ResponseField name="pickupEvents" type="PickupEvent[]">
      Array of pickup event objects
      
      <Expandable title="PickupEvent">
        <ResponseField name="id" type="string (uuid)">
          Pickup event ID
        </ResponseField>
        <ResponseField name="startsAt" type="string (date)">
          Event start time
        </ResponseField>
        <ResponseField name="endsAt" type="string (date)">
          Event end time
        </ResponseField>
        <ResponseField name="location" type="string">
          Pickup location description
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Create Fundraiser

<CodeGroup>
```bash cURL
curl -X POST https://api.curaise.com/api/fundraiser/create \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Spring Food Drive",
    "description": "Help us raise funds for the campus food bank",
    "organizationId": "123e4567-e89b-12d3-a456-426614174000",
    "buyingStartsAt": "2026-04-01T00:00:00.000Z",
    "buyingEndsAt": "2026-04-30T23:59:59.000Z",
    "imageUrls": ["https://example.com/image.jpg"],
    "pickupEvents": [{
      "startsAt": "2026-05-05T10:00:00.000Z",
      "endsAt": "2026-05-05T14:00:00.000Z",
      "location": "Student Center Lobby"
    }]
  }'
```

```typescript TypeScript
const response = await fetch('https://api.curaise.com/api/fundraiser/create', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Spring Food Drive',
    description: 'Help us raise funds for the campus food bank',
    organizationId: '123e4567-e89b-12d3-a456-426614174000',
    buyingStartsAt: '2026-04-01T00:00:00.000Z',
    buyingEndsAt: '2026-04-30T23:59:59.000Z',
    imageUrls: ['https://example.com/image.jpg'],
    pickupEvents: [{
      startsAt: '2026-05-05T10:00:00.000Z',
      endsAt: '2026-05-05T14:00:00.000Z',
      location: 'Student Center Lobby'
    }]
  })
})
```
</CodeGroup>

**POST** `/api/fundraiser/create`

Creates a new fundraiser.

### Authentication

Required - user must be an admin of the specified organization.

### Request Body

<ParamField body="name" type="string" required>
  Fundraiser name (1-255 characters)
</ParamField>

<ParamField body="description" type="string" required>
  Detailed description
</ParamField>

<ParamField body="organizationId" type="string (uuid)" required>
  Organization ID that owns this fundraiser
</ParamField>

<ParamField body="buyingStartsAt" type="string (date)" required>
  ISO 8601 timestamp when buying period begins
</ParamField>

<ParamField body="buyingEndsAt" type="string (date)" required>
  ISO 8601 timestamp when buying period ends
</ParamField>

<ParamField body="imageUrls" type="string[]" required>
  Array of image URLs (must be valid URLs)
</ParamField>

<ParamField body="pickupEvents" type="PickupEvent[]" required>
  At least one pickup event required
  
  <Expandable title="PickupEvent">
    <ParamField body="startsAt" type="string (date)" required>
      Event start time
    </ParamField>
    <ParamField body="endsAt" type="string (date)" required>
      Event end time
    </ParamField>
    <ParamField body="location" type="string" required>
      Pickup location description
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="venmoUsername" type="string" optional>
  Venmo username (5-30 characters)
</ParamField>

<ParamField body="venmoEmail" type="string" optional>
  Email for Venmo account (triggers setup email)
</ParamField>

<ParamField body="goalAmount" type="string (decimal)" optional>
  Target fundraising amount
</ParamField>

### Response

<ResponseField name="message" type="string">
  "Successfully created fundraiser"
</ResponseField>

<ResponseField name="data" type="Fundraiser">
  Created fundraiser object (BasicFundraiserSchema)
</ResponseField>

## Update Fundraiser

**POST** `/api/fundraiser/:id/update`

Updates an existing fundraiser.

### Authentication

Required - user must be an admin of the fundraiser's organization.

### Path Parameters

<ParamField path="id" type="string (uuid)" required>
  Fundraiser ID
</ParamField>

### Request Body

All fields from CreateFundraiserBody except `organizationId` and `pickupEvents`. Pickup events are managed separately.

<ParamField body="name" type="string" required>
  Fundraiser name (1-255 characters)
</ParamField>

<ParamField body="description" type="string" required>
  Detailed description
</ParamField>

<ParamField body="buyingStartsAt" type="string (date)" required>
  When buying period begins
</ParamField>

<ParamField body="buyingEndsAt" type="string (date)" required>
  When buying period ends
</ParamField>

<ParamField body="imageUrls" type="string[]" required>
  Array of image URLs
</ParamField>

<ParamField body="venmoUsername" type="string" optional>
  Venmo username (5-30 characters)
</ParamField>

<ParamField body="venmoEmail" type="string" optional>
  Email for Venmo (sends setup email if changed)
</ParamField>

<ParamField body="goalAmount" type="string (decimal)" optional>
  Target amount
</ParamField>

### Response

<ResponseField name="message" type="string">
  "Successfully updated fundraiser"
</ResponseField>

<ResponseField name="data" type="Fundraiser">
  Updated fundraiser object
</ResponseField>
